Introduction

Python comes with a standard library that provides support for many common programming tasks, such as file I/O, networking, and data manipulation. Additionally, Python allows for the creation of modules and packages, which can be used to organize and share code.

  • Python's extensive standard library includes modules for working with data structures (such as lists, dictionaries, and sets), handling dates and times, and performing mathematical operations.
  • Python can be extended using third-party libraries for specialized tasks. For example, libraries like NumPy and pandas are commonly used for data manipulation and analysis, while TensorFlow and PyTorch are popular for machine learning and artificial intelligence.

Operators

In Python, operators are used to perform operations on variables and values. Python supports a wide range of operators, including arithmetic operators, comparison operators, logical operators, and more.

Python operators work with various data types, such as numbers, strings, lists, and dictionaries.

Here are some examples of operators in Python:

  • Arithmetic operators: + (addition), - (subtraction), * (multiplication), / (division)
  • Comparison operators: == (equal to), != (not equal to), < (less than),> (greater than)
  • Logical operators: and (logical AND), or (logical OR), not (logical NOT)
  • Assignment operators: = (assignment), += (add and assign), -= (subtract and assign)
  • Bitwise operators: & (bitwise AND), | (bitwise OR), ^ (bitwise XOR)

Control Structures
Python ofrece varias estructuras de control para manejar el flujo de un programa. Algunas de las estructuras de control más comunes son:
  • if else: Permite ejecutar bloques de código dependiendo de si una condición es verdadera o falsa.
  • while: Permite ejecutar un bloque de código mientras una condición especificada sea verdadera.
  • for: Permite iterar sobre una secuencia (como una lista o una tupla) y ejecutar un bloque de código para cada elemento.
Ejemplo de uso de estructuras de control en Python: #Ejemplo de if ... else x = 10 if x > 5: print("x es mayor que 5") else: print("x no es mayor que 5") #Ejemplo de while y = 0 while y < 5: print(y) y += 1 #Ejemplo de for for i in range(5); print(i)
Functions

En Python, una función es un bloque de código que solo se ejecuta cuando se llama. Puede aceptar parámetros y devolver valores. Las funciones en Python se definen usando la palabra clave def.

Ejemplo de una función simple en Python:

def greet(name):

return "Hello, " + name

Llamando a la función

print(greet("John"))
Data Structures

Python proporciona varias estructuras de datos incorporadas para almacenar y organizar datos de manera eficiente. Algunas de las estructuras de datos más comunes en Python son:

  • Listas:Una colección ordenda y mutable de elementos.
  • Tuplas:Una colección ordenada e inmutable de elementos.
  • Conjuntos:Una colección no ordenada de elementos únicos.
  • Diccionarios:Una colección de pares clave-valor, donde cada llave es única.

Ejemplo de uso de algunas estructuras de datos en Python: #Lista lista = [1,2,3,4,5] #Tupla tupla = (1,2,3,4,5) #Conjunto conjunto = {1,2,3,4,5} #Diccionario diccionario = {"nombre": "Juan","edad":30}

Modules and Packages

En Python, un módulo es un archivo que contiene definiciones y declaraciones de Python. El nombre del archivo es el nombre del módulo con la extensión `.py`. Un paquete, por otro lado, es una colección de módulos. Puede haber paquetes dentro de paquetes.

Para usar un módulo en Python, primero debe importarlo. Puede importar un módulo completo o solo partes específicas de él.

Ejemplo de importación de un módulo y uso de una función de ese módulo: #Importar el módulo math import math #Usar la función sqrt del móduo math print(math.sqrt(16))

También puede crear sus propios módulos y paquetes en Python para organizar su código de manera más efectiva y reutilizable.

Classes and Objects

En Python, una clase es un plano para crear objetos. Un objeto es una instancia de una clase. Las clases en Python se definen utilizando la palabra clave class

Ejemplo de una clase simple en Python: class Persona: def __init__(self, nombre, edad): self.nombre = nombre self.edad = edad def saludar(self): return "Hola, soy " + self.nombre #Crear un objeto de la clase Persona persona1 = Persona("Juan", 30) #Llamar al método saludar del objeto personal print(persona1.saludar())

En este ejemplo, la clase Persona tiene dos atributos (nombre y edad) y un método (saludar) que devuelve un saludo con el nombre de la persona. Puedes crear tantas instancias de clase Persona como desees.

Exceptions

En Python, una excepción es un evento que interrumpe el flujo normal de ejecución de un programa. Las excepciones se utilizan para manejar errores y otros eventos excepcionales que pueden ocurrir durante la ejecución de un programa.

Ejemplo de uso de excepciones en Python: try: resultado = 10/0 except ZeroDivisionError: print("Error: División por cero") else: print("La división fue exitosa") finally: print("Finalizando..")

En este ejemplo, el bloque de código dentro del try intenta dividir 10 por 0, lo cual generará una excepción ZeroDivisionError. El bloque except maneja esta excepción, imprimiendo un mensaje de error. El bloque else se ejecuta si no se produce ninguna excepción, y el bloque finally se ejecuta siempre, independientemente de si se produjo una excepción o no.

File Input and Output

En Python, puede trabajar con archivos para leer y escribir datos. Puede abrir un archivo utilizando la función open() y luego usar métodos como read(), write() y close() para interactuar con él.

Ejemplo de lectura y escritura básica de un archivo en Python: #Abrir un archivo en modo escritura file = open("archivo.txt","w") # Escribir en el archivo file.write("Hola, mundo!") # Cerrar el archivo file.close() # Abrir el archivo en modo lectura file = open("archivo.txt", "r") # Leer el contenido del archivo contenido = file.read() print(contenido) # Cerrar el archivo file.close()

También puede usar la declaración with para abrir un archivo. Esta declaración asegura que el archivo se cierre correctamente una vez que haya terminado de usarlo, incluso si ocurre una excepción durante la operación.

Ejemplo de uso de la declaración with para leer un archivo en Python: with open("archivo.txt", "r") as file: contenido = file.read() print(contenido)
